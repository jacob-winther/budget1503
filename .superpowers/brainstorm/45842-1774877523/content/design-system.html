Animation System Foundation

CSS custom properties that govern every transition in the app

Timing & Easing

--ease-out-expo
cubic-bezier(0.16, 1, 0.3, 1)
Fast start, glides to rest. Used for entrances.
--ease-in-expo
cubic-bezier(0.7, 0, 0.84, 0)
Slow start, fast exit. Used for exits.
--dur-enter
280ms
All entrance animations. Balanced feel.
--dur-exit
200ms
Exits slightly faster — less drag on workflow.
--dur-micro
150ms
Hovers, highlights, small state changes.
--stagger-row
30ms per row
Delay multiplier for cascading row entrances.

What this unlocks

All 7 animation moments are built from these 5 variables. Changing --dur-enter in one place adjusts every entrance in the app. This keeps the feel consistent and makes future tuning trivial.

/* src/style.css — added to :root */
:root {
  --ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1);
  --ease-in-expo: cubic-bezier(0.7, 0, 0.84, 0);
  --dur-enter: 280ms;
  --dur-exit: 200ms;
  --dur-micro: 150ms;
  --stagger-row: 30ms;
}
