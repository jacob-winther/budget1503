The 7 Animation Moments

How each interaction behaves — scroll to see all

A · Expand / Collapse

Before
Rows appear/disappear instantly. Table jumps.
After
grid-template-rows animates 0fr → 1fr (CSS height trick, no JS). Child rows stagger in with 30ms delay each. Chevron rotates 90° with --dur-micro.
Vue component: BudgetSectionRow + BudgetCategoryRow — wrap children in <TransitionGroup name="row">

B · Adding a new row

Before
Row pops in at full size immediately.
After
Slides up from 8px below + fades in over 280ms ease-out-expo. A brief left-border highlight (--accent color) flashes for 600ms to draw the eye.
Vue component: BudgetItemRow / BudgetCategoryRow — .row-enter-active CSS class

C · Deleting a row

Before
Row vanishes. Table reflows with a jump.
After
Fades out + slides left 12px over 200ms ease-in-expo. Height collapses to 0 after opacity reaches 0, so the table reflows smoothly.
Same TransitionGroup — .row-leave-active + position:absolute during leave so siblings animate into place

D · Edit mode enter / exit

Before
Text swaps to input instantly.
After
Row background transitions to a subtle highlight shade over 150ms. Inputs fade in. On save/cancel, reverse. The active edit row gets a left-border accent glow.
Vue component: BudgetItemRow — CSS transition on background-color + <Transition name="fade"> around inputs

E · Totals updating

Before
Numbers snap to new value silently.
After
A useAnimatedNumber composable tweens the displayed value over 300ms ease-out-expo using requestAnimationFrame. No library needed — ~20 lines.
New file: src/composables/useAnimatedNumber.ts — used in BudgetTotalsRow + BudgetDifferenceRow

F · Dialog open / close

Before
Dialog snaps in with no entrance.
After
Dialog panel scales from 0.95 → 1 + fades in over 280ms ease-out-expo. Backdrop fades separately at 200ms. Exit reverses at 200ms.
Vue components: ItemFormDialog, CategoryFormDialog, CopyYearDialog — <Transition name="dialog"> wrapper

G · Year switching

Before
Table replaces instantly on year change.
After
Table slides out in the direction of navigation (← for prev year, → for next) and new year slides in from opposite side. Uses <Transition :name="slideDirection"> with a dynamic name based on whether year increased or decreased.
Vue component: App.vue — wrap <BudgetTable> in <Transition :name="yearSlideDir">, track direction in budgetStore
